# 🛒 Retail Point of Sale (POS) & Inventory Management System

A web-based Point of Sale (POS) and Inventory Management System built from scratch using PHP and MySQL. This project is designed to automate retail sales transactions, compute totals in real-time, and track product stock levels dynamically to minimize human error in inventory handling.

## 🚀 Key Features

- **Secure Authentication:** Built-in login and logout system with role-based access for Admins and Cashiers to secure business data.
- **Sales & Checkout Interface:** Allows quick item additions to the cart, with real-time calculations for Grand Total, Tax, Discounts, and Customer Change.
- **Real-Time Inventory (CRUD):** - Complete capability to Create, Read, Update, and Delete product records.
  - Automatic stock deduction from the database upon every successful sales transaction.
- **Transaction History & Logs:** Stores and tracks all previous sales records, allowing Admins to review data easily for auditing and sales reporting.

---

## 🛠️ Tech Stack Used

- **Front-End:** HTML5, CSS3 (Fully Responsive Layout), JavaScript
- **Back-End:** PHP (Procedural/OOP Core Logic)
- **Database:** MySQL (Relational Database Management)
- **Server Environment:** XAMPP (Apache)
